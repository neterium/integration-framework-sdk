# Security

If you believe you have found a security vulnerability in this repository, please report it privately

Email: security@neterium.com

Please do not disclose security issues publicly (e.g., via public issues) until Neterium has had a reasonable opportunity to assess them.
Neterium does not commit to any specific response or remediation timelines for reported vulnerabilities.

If private vulnerability reporting is enabled for this repository, you may also report vulnerabilities via GitHub’s Private Vulnerability Reporting feature.
